# Business Style

Use for:
- 汇报老板
- 客户方案
- 商务介绍
- 正式内部汇报

## Traits

- calm, trustworthy, professional
- restrained color palette
- stronger information hierarchy
- fewer decorative elements
- suitable for charts, comparisons, roadmaps, summaries

## Recommended colors

- background: white / light gray / navy
- primary: deep navy, dark blue, muted teal
- accent: subtle blue or red for emphasis only

## Image direction

Prompt hints:
- 商务风
- 企业感
- 稳重
- 简洁
- 专业汇报视觉

Avoid:
- cartoon style
- overly playful elements
- too many floating abstract shapes

## Layout preference

Prefer:
- hero cover
- 2-column explanation
- comparison page
- 3-card summary
- timeline / roadmap
- conclusion page
