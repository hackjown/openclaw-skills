# Dark Style

Use for:
- 技术方案
- AI / agent / automation / platform topics
- 发布感更强的演示
- 需要更强科技感的 PPT

## Traits

- dark background
- strong contrast
- modern tech feeling
- slightly more dramatic than training/business style
- still readable first, visual second

## Recommended colors

- background: deep navy, charcoal, midnight blue
- primary text: white / light gray
- accent: cyan, electric blue, purple, orange in small doses

## Image direction

Prompt hints:
- 暗黑科技风
- 深色背景
- 高对比
- 科技感
- 简洁专业

Avoid:
- dense neon overload
- unreadable low-contrast text
- cluttered cyberpunk scenes

## Layout preference

Prefer:
- hero cover
- focus page with large image
- architecture / workflow page
- card grid
- metric / benefit summary
- strong final takeaway
